## Running the Project

Install dependencies:

```bash
npm i
```

Start the development server:

```bash
npm run dev
```

---

## Tech Stack

- React/Typescript  
- Vite  
- Sanity (Headless CMS)

---

## Notes

If you encounter dependency issues:

```bash
rm -rf node_modules package-lock.json
npm install
```
